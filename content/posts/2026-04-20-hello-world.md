# Hello World

Welcome to my new site! This is built with [Jaspr](https://jaspr.site), a Dart web framework for building fast, server-rendered web apps.

## Why Jaspr?

Jaspr lets you write your entire web app in Dart — server rendering, client hydration, routing, and styling all in one language. No JavaScript required.

```dart
// A simple Jaspr component
class HelloWorld extends StatelessComponent {
  @override
  Component build(BuildContext context) {
    return p([.text('Hello, world!')]);
  }
}
```
